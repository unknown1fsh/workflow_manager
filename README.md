# workflow_manager
Workflow Manager, iş süreçlerinizi dijitalleştirip optimize eden kurumsal bir platformdur. Spring Boot ve Angular 17 ile güvenli, ölçeklenebilir yapı sunar. Adım, durum ve görev yönetimini kolaylaştırır. Design, automate and monitor workflows with JWT auth, RBAC and real-time updates. Modern PrimeNG UI delivers a sleek, responsive UX for all teams.
